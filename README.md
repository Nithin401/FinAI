# FinAI | Intelligent Wealth Management Dashboard 🚀

FinAI is a modern Fintech application designed to democratize financial literacy through Artificial Intelligence. The platform features a neural investment agent that analyzes user demographics and income to provide a tailored wealth-building roadmap.



## 🌟 Key Features

* **AI Investment Agent:** Uses a proprietary logic engine to calculate optimal asset allocation (Stocks, Gold, Insurance) based on Age, Monthly Savings, and Yearly Revenue.
* **Smart Dashboard:** A clean, glassmorphism-inspired UI for tracking "Total Savings" and "Auto-Debit" schedules.
* **Dynamic Fund Management:** Interactive "Add Funds" functionality with real-time UI updates.
* **Professional Landing Page:** High-conversion hero section and integrated authentication gateway.
* **Persistent State:** Utilizes Browser Local Storage to maintain user balance and scheduling data.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure for high SEO and accessibility.
* **CSS3:** Custom properties (CSS variables), Flexbox, and Grid for a responsive, dark-mode aesthetic.
* **JavaScript (ES6+):** Pure Vanilla JS for AI calculations, DOM manipulation, and state management.

## 🧠 The AI Logic

The **FinAI Strategist** uses a modified version of the "Rule of 100" and the 50/30/20 budgeting principle. It calculates risk tolerance based on the user's age:

* **Aggressive:** Higher stock allocation for users under 30.
* **Conservative:** Higher insurance and gold allocation for users over 45.
